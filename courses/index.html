Summary of courses related to IT taken at the Illinois Institute of Technology

    ITM 301

  • Introduction to OS and Hardware
  • Fall 2017

    • Course Content:

  • System repair (software and hardware components)
  • Troubleshooting
  • Installation processes (software, hardware, and OS)

    ITM 311

  • Introduction to Software Development
  • Fall 2017

    • Course Content:

  • Object-oriented programming (focus on JAVA)
  • Proper coding syntax
  • Uses of GUI's
  • Final project consisted of coding an original program

ITMO 456

Introduction to Open Source Operating

Content was focused on the installation, basic administration, command-line commands related to open-source operating systems. Primary focus was on the Fedora and Ubuntu distributions of Linux.

ITMM 471

Project Management for Information Technology and Management

The course covered the basics of project management, particularly planning projects that involve hardware, softare, and network implementation.

ITMO 440

Introduction to Data Networks and the Internet

Introduction to data networks including network components, Internet architecture, protocols, and network tehcnologies.
